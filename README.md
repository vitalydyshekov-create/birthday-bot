# Birthday Bot 🍺🍗

Telegram-бот для привітань з днем народження у групі.

## Команди

- `/add Ім'я 25.07` — додати людину
- `/list` — показати список
- `/remove Ім'я` — видалити
- `/test` — перевірити роботу
- `/chatid` — дізнатися ID групи

## Render

Build Command:
```
pip install -r requirements.txt
```

Start Command:
```
python bot.py
```

Environment Variables:
```
BOT_TOKEN = ваш новий токен
CHAT_ID = ID вашої групи
```
